# 홈앤쇼핑 통합 포탈 도구
## Product Requirements Document (PRD)

---

## 1. 목표 (무엇을, 왜 만드는가)

### 핵심 목표
직원들이 **하나의 포탈에서** 매출 분석, 관련 사이트 접근, 업무 지원을 한번에 받을 수 있는 통합 대시보드 구축

### 왜 필요한가
- 여러 사이트에 흩어져 있는 정보를 찾아다니는 시간 낭비 제거
- 영업팀, MD팀을 포함한 전 직원이 **동일한 매출 데이터**를 실시간으로 확인
- 직관적인 매출 분석으로 빠른 의사결정 지원
- 검색/질문 기능으로 업무 효율성 향상

---

## 2. 주요 사용자

| 사용자군 | 주요 활용 용도 |
|---------|-------------|
| **영업팀** | 일일 매출 확인, 목표 달성도 모니터링 |
| **MD팀** | 월별/년도별 실적 분석, 상품 전략 수립 |
| **경영진** | 목표 대비 달성률 한눈에 파악 |
| **일반 직원** | 업무 관련 사이트 빠른 접근, 궁금한 점 검색 |

## 3. 핵심 기능 (MVP 중심, 단순 구현)

### 3.1 대시보드 (메인 화면) ⭐ 핵심
포탈 진입 시 보이는 메인 화면
- **헤더**: 홈&쇼핑 포탈 로고 + 현재 시간 표시
- **메인 콘텐츠**: 버튼/카드 형식으로 그룹화된 섹션
  - 📊 **매출 분석** (가장 크게 강조)
  - 🔗 시스템 링크 그룹 (BOS, 그룹웨어, 콜판 등)
  - 💬 Claude 챗 기능
  - ⚙️ 설정 & 도움말
- **푸터**: 저작권 정보

### 3.2 매출 분석 섹션 (MVP 핵심 기능 ⭐⭐⭐)
**메인 분석 화면** (클릭 시 상세 페이지 이동)

**일별 실적 조회**
- 표 형식으로 최근 7일간의 일일 매출 표시
- 컬럼: 날짜 | 매출액 | 전날 대비 증감률(%)
- 오늘 매출을 가장 위에 강조 표시
- 컬러 코딩: 증가(초록), 감소(빨강)

**월별 실적 조회**
- 현재 월 누적 매출 (큰 폰트로 강조)
- 월 목표액 표시
- 목표 대비 달성률 (%) - 프로그레스 바로 시각화
- 월별 비교 표: 지난 3개월 (4월 ~ 6월) 매출 & 달성률
- 월별 전년 동월 대비 성장률 표시

**년도 목표 현황**
- 년도 목표액: ○○억원
- 현재까지 누적(6개월): ○○억원
- 년도 목표 달성률 (%) - 프로그레스 바
- 남은 기간(6개월)에 필요한 평균 월 매출액 자동 계산

**목표 관리 기능**
- 월 목표액 입력/수정 가능 (input 창)
- 년도 목표액 입력/수정 가능
- 목표값은 로컬 스토리지에 저장 (새로고침해도 유지)
- 달성률 자동 계산 함수 (누적 매출 / 월 목표 × 100)

**데이터 출처**
- 초기 v1: 목업 데이터 (가상의 매출 데이터)
- 향후 v2: 실제 시스템(OLAP, DB) 연동

### 3.3 시스템 링크 그룹 (기존 포탈 유지)
**카테고리별 버튼/카드 그룹**

**홈쇼핑 플랫폼**

### 3.4 Claude 챗 기능 (보조 기능)
간단한 검색/질문 기능
- 질문 입력창 제공
- Claude API 연동하여 업무 관련 질문에 답변
- 예시 질문:
  - "월 목표는 언제까지인가?"
  - "최근 7일 매출 추이는?"
  - "월별 성장률이 어떻게 되나?"
  - "목표 달성을 위해 남은 기간의 필요 매출액은?"

### 3.5 설정 & 도움말 (보조 기능)
**설정 페이지**
- 사용자 프로필 (직급, 부서, 이름)
- 월 목표액 설정
- 년도 목표액 설정
- 매출 분석 데이터 새로고침 (수동 갱신)

**도움말 & 접속 정보**
- 각 시스템별 접속 ID/PW (보안 주의)
- 매출 분석 사용 설명서
- FAQ

### 3.6 레이아웃 유지
기존 포탈 스타일 그대로 유지
- 심플하고 직관적인 버튼/카드 레이아웃
- A형 / B형 선택지 제공 (선택사항)
- 빠른 네비게이션 (원클릭으로 각 시스템 접근)
- 게임 기능 (너구리, 테트리스) - 선택사항

## 4. Out of Scope (v1 제외, 향후 고도화 시 추가)

### 4.1 사용자 관리 & 권한
**제외 이유**: 모든 직원이 동일한 데이터를 봐야 하기 때문
- 사용자별 로그인 & 인증 (현재는 누구나 접근 가능)
- 직급/부서별 권한 관리
- 개인화된 대시보드 (모두 동일한 화면)
- 사용자별 조회 기록/히스토리 추적

**v2 고도화**: 관리자 권한, 데이터 접근 제한 등

### 4.2 상품/카테고리별 분석
**제외 이유**: 데이터 복잡도 증가, 1일 완성 어려움
- 상품 카테고리별 매출 분석
- 상품군별 성장률 비교
- 상품별 TOP 10 순위
- 카테고리별 목표 vs 실적

**v2 고도화**: 드릴다운 분석 (전사 → 부문 → 상품)

### 4.3 방송 편성표 & 콘텐츠 관리
**제외 이유**: 별도 시스템(방송 심의 시스템) 연동 필요
- 방송 편성 시간표
- 라이브 방송 스케줄
- 콘텐츠 기획표
- 진행자 스케줄

**v2 고도화**: 방송심의 시스템(Beta Live)과 API 연동

### 4.4 알림 & 공지사항
**제외 이유**: 푸시 기능 및 알림 인프라 필요
- 매출 목표 미달 시 알림
- 중요 공지사항 팝업
- 실시간 알림 (메일, SMS)
- 공지사항 게시판

**v2 고도화**: 알림 시스템 구축, 중요도별 필터링

### 4.5 모바일 앱
**제외 이유**: 웹 기반으로만 제공하여 개발 간소화
- 네이티브 iOS/Android 앱
- 모바일 푸시 알림
- 오프라인 모드
- 모바일 전용 UI/UX

**v2 고도화**: 반응형 웹 디자인, PWA(Progressive Web App) 추가

### 4.6 데이터 내보내기
**제외 이유**: 화면 조회만으로 충분, 파일 생성 기능 생략
- Excel 다운로드 (.xlsx)
- PDF 레포트 생성
- CSV 데이터 추출
- 정기 자동 리포트 메일

**v2 고도화**: 원클릭 Excel 내보내기, 정기 리포트 자동화

### 4.7 고급 분석 기능
**제외 이유**: 기본 분석에만 집중
- 시계열 예측 (미래 매출 추정)
- 이상 탐지 (매출 급변 감지)
- 요일별/시간대별 분석
- 고객 세그먼트 분석

**v2 고도화**: 머신러닝 기반 고급 분석

### 4.8 데이터 동기화 & 실시간 업데이트
**제외 이유**: 초기 v1은 수동 갱신
- 실시간 매출 데이터 자동 동기화
- 다중 데이터 소스 통합 (ERP, POS 등)
- 자동 새로고침 (polling/WebSocket)
- 데이터 캐싱 및 최적화

**v2 고도화**: 실제 DB 연동, API 기반 자동 갱신

### 4.9 국제화 & 다국어
**제외 이유**: 국내 직원 대상
- 다국어 지원 (영어, 중국어 등)
- 다중 통화 (KRW, USD 등)
- 지역별 시간대 처리

**v2 고도화**: 필요시 추가

### 4.10 보안 강화
**제외 이유**: MVP에서는 기본 보안만
- 2단계 인증 (2FA)
- 역할 기반 접근 제어 (RBAC)
- 감사 로그 (Audit Log)
- 암호화된 데이터 전송
- IP 화이트리스트

**v2 고도화**: 엔터프라이즈급 보안 기능 추가
